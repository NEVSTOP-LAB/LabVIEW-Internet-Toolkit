Introduction

Welcome to the CGI examples that are part of the Internet Toolkit.

The CGI examples must be used with the G Web Server (HTTP Server) which can be run in LabVIEW 7.0 or later.

Some of the examples use server-push animations which can be currently only viewed in Netscape Navigator.

Important!

The CGI examples will not work unless you are retrieving the pages through the G Web Server. If the current page's URL in the Location field in your browser starts with file:///, then you are not using the server. First make sure your server is running and it is configured to point to the internet/home directory. Then go to the URL http://your.computer.name/, where your.computer.name is the Internet address of your computer. If you do not know what your computer's address is, you can use the localhost address 127.0.0.1.
