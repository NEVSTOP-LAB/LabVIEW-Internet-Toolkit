One Shot DAQ

This CGI generates an HMTL form that prompts you for a channel, number of samples and a sampling rate. When you submit the form, the CGI acquires the data and returns a form with an embedded image identifying the data (using a cookie).

When the CGI is called with a Magic_Cookie parameter, it returns an image of the data associated with the cookie.

Note that by default, this vi is in a demo mode. If you have a DAQ Board (Device 1) you can change the vi to non-demo mode. To change the vi to non-demo mode, change the Boolean constant in the daqcgi.vi labeled demo mode to false.
