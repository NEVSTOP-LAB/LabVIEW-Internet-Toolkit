The following examples demonstrate different capabilities of the G Web Server. You can use them as the starting points when developing your own pages or CGI VIs.

Access Control
Limiting access to documents and images based on user and password.

Panel Images
Publishing static and animated images of VI panels.

CGI Basics
How data is passed to and returned from CGI VIs.

CGI Examples
Advanced CGI application examples. Examples include Building HTML Documents, Executing VIs, Controlling Other VIs, Maintaining State Information, and Returning Panel Images.

Image Maps
Using image maps to respond to user clicks.

Sample Application
Examples that demonstrate more complicated CGI applications.

Build the G Web Server Into an Executable
Steps for building an executable out of the G Web Server that uses CGI VIs and an example of the build instructions on one of the shipping examples in the toolkit.

Creating Simultaneous Instances Remote Panels LabVIEW Applications
Using the CGI scripting to dynamically access the LabVIEW VI Server and return individual instance of the template VI.
