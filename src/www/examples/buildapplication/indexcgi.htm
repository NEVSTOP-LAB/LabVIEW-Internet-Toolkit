Introduction

This is a sample page to illustrate the use of the G Web Server in a built application.

Important!

The CGI examples will not work unless you have configured the srm.cfg file to have the correct DocumentRoot for this built application. For example, the DocumentRoot for this built application should be <LabVIEW Dir>/www/examples/buildapplication/standalone/www or <LabVIEW Dir>/www/examples/buildapplication/independent/www.
