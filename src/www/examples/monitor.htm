Panel Image Animations

The following is an embed image of a server-push animation of the front panel of the VI Waveform Example.vi. The image updates approximately once a second for the duration of 60 seconds. The HTML code for this image is

<IMG src="/.monitor?Waveform%20Example.vi&refresh=1&lifespan=60">

Note that the animation only works in browsers that support server-push animation, such as Netscape Navigator. Other browsers simply receive a static image.
